About this website

This website uses CSS and HTML only and was written almost entirely with the program Vim. Its DOCTYPE is HTML5, even though no HTML5 features are used here. In a technical sense, the goal of this website is to keep it small, simple and therefore, make the content load in a fast and reliable manner. It doesn't use any CGI scripts or JavaScript. It's hosted by GitLab Pages.
